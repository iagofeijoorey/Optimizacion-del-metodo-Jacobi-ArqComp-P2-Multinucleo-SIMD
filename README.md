# Optimizaci-n-del-m-todo-Jacobi-ArqComp-P2-Multin-cleo-SIMD
Implementación y análisis de rendimiento del método iterativo de Jacobi en C, explorando cuatro niveles de optimización: versión secuencial base, optimización de caché , paralelismo con OpenMP y vectorización SIMD con AVX2/FMA. Mediciones realizadas en el supercomputador FinisTerrae III para distintos tamaños de matriz.
